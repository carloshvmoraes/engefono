### **DeglutiSom**

Software de análise de sons da deglutição.
O Software DeglutiSom é um software gratuito de uso restrito, o qual é disponibilizado através da participação no treinamento de Análise de Sons da Deglutição. [Ver mais.](https://www.engefono.com.br/?page_id=29 "Treinamentos")
[caption id="attachment\_219" align="alignnone" width="955"][![DeglutiSom e Doppler](https://www.engefono.com.br/wp-content/uploads/2015/07/DeglutiSom_Doppler_Exemplo-1024x947.jpg)](https://www.engefono.com.br/wp-content/uploads/2015/07/DeglutiSom_Doppler_Exemplo.jpg) Programa DeglutiSom e Doppler[/caption]
Requerimentos mínimos:

* Sistema Operacional: Microsoft Windows Vista, Windows  7, Windows  8, Windows 8.1 ou Windows 10 nas versões 32 ou 64 bits.
* Memória RAM: 2 GBytes devido a gravação de som.
* Disco rígido (HD): 600 MBytes.
* Línguas: Português, Inglês e Espanhol.

As versões Android e iOS ainda não estão disponíveis.
O Software DeglutiSom está registrado no INPI.
Dúvidas no uso do DeglutiSom: [utilize nossa página de suporte](https://www.engefono.com.br/?page_id=295).
**O DeglutiSom ainda não é suportado por iOS, Mac, Linux e Windows Phone.**
