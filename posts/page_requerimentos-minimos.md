O DeglutiSom necessita dos seguintes requerimentos

* Windows XP, Windows Vista, Windows 7, Windows 8, Windows 8.1 ou Windows 10.
* 2GBytes de Memória RAM
* 700 Mbytes de disco
* Microsoft Framework 4.0 (já vem no pacote de instalação do programa).

**Atenção: Não é compatível com Windows Phone, Linux, Android, iOS ou Mac.**
