Esta página descreverá em passos como acessar e alterar as configurações de placa de som do computador, Tablet Windows ou notebook.
Problemas relacionados a gravação de som ambiente ou mesmo não obter gravação alguma.
Sugerimos que, mesmo seguindo estes passos o [problema não for solucionado](#NaoSolucionado), procure o suporte técnico do fabricante do dispositivo (Dell, HP, Asus, Lenovo) devido a possibilidade de ser uma falha de hardware ou driver da placa de som.

## Acessando a Janela de Configurações de Som do Windows

Para acessar a janela de configurações de som do Windows, pressione o botão DIREITO do mouse sobre o ícone de áudio, próximo ao relógio (canto inferior direito).
[caption id="attachment\_419" align="alignnone" width="463"][![AcessoSomGravacao](https://www.engefono.com.br/wp-content/uploads/2015/11/AcessoSomGravacao.jpg)](https://www.engefono.com.br/wp-content/uploads/2015/11/AcessoSomGravacao.jpg) Acessando janela de Som do Windows diretamente.[/caption]
Selecione a opção gravação do menu exibido.
[caption id="attachment\_420" align="alignnone" width="374"][![OpcaoGravacao](https://www.engefono.com.br/wp-content/uploads/2015/11/OpcaoGravacao.jpg)](https://www.engefono.com.br/wp-content/uploads/2015/11/OpcaoGravacao.jpg) Selecionando as entradas de gravação de som.[/caption]
A janela de gravação de som será apresentada. Nela são encontrados todas as entradas de som funcionais. No canto de cada entrada existe um nível de volume. Esse nível ajudará a identificar qual entrada é o microfone do notebook/tablet/computador e qual é a entrada externa do Doppler.
[caption id="attachment\_421" align="alignnone" width="604"][![GravacaoNiveis](https://www.engefono.com.br/wp-content/uploads/2015/11/GravacaoNiveis.jpg)](https://www.engefono.com.br/wp-content/uploads/2015/11/GravacaoNiveis.jpg) Janela de entradas para a gravação de som, os níveis apresentados permitem identificar se o Doppler encontra-se ligado a alguma delas.[/caption]
**Atenção:  As imagens e descrições das entradas de som podem variar conforme modelo ou fabricante do computador. A imagem acima foi obtida de um desktop Lenovo.**

## Entradas Ocultas

As entradas de som desabilitadas ou desativadas, por padrão, não são exibidas na janela de gravação. Desta forma, é importante visualizá-las caso o Doppler não for encontrado nas entradas de som exibidas na janela de gravação do Windows inicial.
Para exibir essas entradas, pressione o botão DIREITO do mouse sobre a área branca abaixo das entradas visíveis da janela de gravação do Windows, conforme figura.
[caption id="attachment\_422" align="alignnone" width="437"][![ExibeMenuOculto](https://www.engefono.com.br/wp-content/uploads/2015/11/ExibeMenuOculto.jpg)](https://www.engefono.com.br/wp-content/uploads/2015/11/ExibeMenuOculto.jpg) Botão direito do mouse exibe menu oculto na janela de gravação de som do Windows. (os tipos ou imagens dos dispositivos podem variar)[/caption]
Com isso será exibido um menu oculto. Selecione ambas as opções desse menu para visualizar os dispositivos desabilitados e desconectados.
[caption id="attachment\_423" align="alignnone" width="483"][![MenuOcultoSom](https://www.engefono.com.br/wp-content/uploads/2015/11/MenuOcultoSom.jpg)](https://www.engefono.com.br/wp-content/uploads/2015/11/MenuOcultoSom.jpg) Menu para exibir entradas de som desabilitadas ou desconectadas.[/caption]
(os tipos ou imagens dos dispositivos podem variar)
Assim, novas entradas serão exibidas na janela de gravação de som. Com a possibilidade de uma delas ser a ligada a entrada externa de som do Doppler.

# Ainda não consigo gravar o Doppler!

Em alguns casos, mesmo realizando os passos descritos acima, não é identificada a entrada externa de som do computador.
Isso pode ser decorrente a dois problemas:

### Adaptador Incompatível para Computador

No caso das entradas de HeadSet, o adaptador comprado para o Doppler pode não ser compatível para o computador. Já ocorrendo casos o adaptador ser incompatível para notebooks do mesmo fabricante.
Consulte o [manual ou a página de suporte técnico do fabricante](https://www.engefono.com.br/?page_id=430) para obter o modelo do adaptador de som compatível para o computador, Tablet ou notebook.

### Driver da Placa de Som Desatualizado

Verifique no gerenciador de dispositivos do Windows se o driver da placa de som está funcionando corretamente.
Mantenha sempre os drivers de som atualizados, sempre [consulte a página de suporte técnico do fabricante](https://www.engefono.com.br/?page_id=430), pois drivers antigos podem apresentar problemas que serão corrigidos apenas instalando uma nova versão existente no site do fabricante.

## Solução Alternativa

Uso de uma placa de som USB para notebooks e computadores. A maioria dos Tablets não possui entrada USB, apenas saída, para ligar esta placa de som.
[Veja aqui como utilizar a placa de som USB.](https://www.engefono.com.br/?page_id=442)
